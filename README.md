# Interview IQ 🎙️🤖  
_3rd Place Winner – MadHacks 2025_

**Interview IQ** is an AI-powered real-time interview preparation platform that provides instant, personalized feedback across technical, behavioral, and system design interviews. Built for job seekers, students, and professionals looking to refine their communication, body language, and technical articulation.

---

## 🔥 Features

- 🎥 **Live Video Analysis**  
  Tracks facial expressions, eye contact, and posture using OpenCV.

- 🧠 **AI-Powered Response Scoring**  
  Evaluates answers for relevance, structure, and fluency via Google Gemini Pro.

- 🎙️ **Speech Analysis**  
  Detects filler words, speaking pace, tone, and clarity.

- 📊 **Performance Dashboard**  
  Visual feedback and session comparisons to track improvement over time.

- 🔁 **Multiple Interview Modes**  
  Supports behavioral, technical, and system design scenarios.
